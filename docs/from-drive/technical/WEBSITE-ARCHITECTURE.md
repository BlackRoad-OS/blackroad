7.5 Tech Stack Recommendation
Framework: Next.js 14 (App Router)
Styling: Tailwind CSS + shadcn/ui
Auth: Clerk or NextAuth
Database: Supabase (PostgreSQL)
Payments: Stripe
Hosting: Vercel (frontend) + Railway (backend)
CDN/DNS: Cloudflare
Real-time: Socket.io or Supabase Realtime
AI: Your existing multi-model setup
This architecture provides a unified ecosystem where users can flow naturally between products while maintaining context. The hub-and-spoke model with blackroad.io as the center ensures discoverability while allowing each product to have its own focused domain.
